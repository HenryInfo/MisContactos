Bienvenido a la página GitHub del Proyecto 'Mis Contactos'

Compartimos el código del proyecto utilizado en el tutorial "Desarrollo en Tu Idioma: AndroidStudio". En cada uno de los capítulos vamos realizando commits al proyecto para cada etapa importante en el ciclo de desarrollo del mismo. Te invitamos a descargarlo, ya sea utilizando tus herramientas preferidas para el control de versiones GIT, descargando el proyecto en ZIP, o bien utiliza AndroidStudio para la clonación de proyectos GitHub que tiene integrado.
